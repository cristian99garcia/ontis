Ontis
---

A lightweight browser written in Vala + WebKit 3.0

![ontis_screenshot](https://raw.githubusercontent.com/cristian99garcia/ontis/master/screenshots/screenshot1.png)


For run this:
First compile with:
```make```

And after run with:
```./ontis```

*Dependencies:*

 * GTK3 (>= 3.10)
 * WebKit (3.0)
 * libsoup (2.4)
 * JsonGLib (1.0)
 * valac
